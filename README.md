## Setup

1. Install dbt:
   ```bash
   pip install dbt-databricks
   ```

2. Copy `profiles.yml` to `~/.dbt/profiles.yml` and update with your Databricks credentials:
   - `host`: Your Databricks workspace URL
   - `http_path`: SQL warehouse HTTP path
   - `token`: Personal access token
   - `catalog`: Target catalog (default: `workspace`)

3. Run dbt:
   ```bash
   cd pilot_testdb_databricks
   dbt deps
   dbt run
   dbt test
   ```
